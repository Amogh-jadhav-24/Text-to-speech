# Text-to-speech

This is an Text-to-Speech Webpage Where You can Convert your Words to Voices!

The System is made by using Html Css And Javascript With a SpeechSynthesisUtterance() 


The SpeechSynthesisUtterance() interface of the Web Speech API represents a speech request. It contains the text the speech service should read aloud and parameters for how to read it, such as language, pitch, volume, and speaking rate
